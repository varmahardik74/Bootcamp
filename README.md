# Bootcamp

This repository contains my practice work and lecture code from the Web Development Bootcamp.


## Topics Covered

- HTML and CSS basics
- JavaScript fundamentals
- Functions and file handling practice
- Promises and asynchronous JavaScript
- DOM manipulation
- CRUD operations with the DOM
- HTTP basics with Node.js

## How to Use

Clone the repository:

```bash
git clone https://github.com/varmahardik74/Bootcamp.git
```

Move into the project folder:

```bash
cd Bootcamp
```

Open the lecture folders and run the examples as needed.

For HTML/CSS/DOM lectures, open the `index.html` file in your browser.

For Node.js based lectures, move into the lecture folder and install dependencies:

```bash
cd "Web-Dev Cohort/Lec-7&8(HTTP)"
npm install
```

Then run the JavaScript file:

```bash
node index.js
```
